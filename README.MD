# Stack
 - Kotlin
 - Timber & LeakCanary
 - Material 2
 - Lottie
 - Retrofit & OkHttp
 - Kotlinx serialization
 - Hilt/Dagger
 - Jetpack navigation components
 - Room
 - Google play services
 - JUnit
 - JodaTime
